## ğŸ“– Logik Perhitungan

Algoritma ini secara amnya menggunakan struktur kawalan `if-else` atau `switch-case` untuk menentukan komponen gaji:

1. **Gaji Pokok**: Ditentukan mengikut kategori/golongan pekerja.
2. **Tunjangan**: Ditambahkan ke gaji pokok (contoh: 10% daripada gaji pokok).
3. **Pajak**: Potongan peratusan tertentu daripada gaji kasar.
4. **Gaji Bersih**: `(Gaji Pokok + Tunjangan) - Potongan`.
