Repositori ini mengandungi pelaksanaan algoritma ringkas untuk mengira gaji bersih pekerja berdasarkan beberapa parameter seperti gaji pokok, elaun, dan potongan. Projek ini sangat berguna bagi pelajar yang ingin memahami cara menterjemahkan logik perniagaan ke dalam kod program.

ðŸ“Œ Ciri-ciri Utama
Input Data: Mengambil data nama pekerja dan golongan.

Logik Pengiraan:

Menentukan Gaji Pokok berdasarkan golongan.

Mengira Tunjangan (Elaun) seperti tunjangan keluarga atau jabatan.

Mengira Potongan (Pajak/Insurans).

Output: Menampilkan slip gaji ringkas di konsol.

ðŸ“– Logik Perhitungan
Algoritma ini secara amnya menggunakan struktur kawalan if-else atau switch-case untuk menentukan komponen gaji:

Gaji Pokok: Ditentukan mengikut kategori/golongan pekerja.

Tunjangan: Ditambahkan ke gaji pokok (contoh: 10% daripada gaji pokok).

Pajak: Potongan peratusan tertentu daripada gaji kasar.

Gaji Bersih: (Gaji Pokok + Tunjangan) - Potongan.
